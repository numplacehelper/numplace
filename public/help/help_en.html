How to Play Namber Place (Sudoku)

Rule

Putting numbers from one to nine onto the empty cells of the board so that same numbers never appear in a row, a column, and a block. In other words, every row (column and block) has all numbers from one to nine.

Playing with the number place helper

Starting a game

Playing same games

Pressing a button below the “Sample Games”. From the left, “B” stands for begginer, “E” for easy, “M” for medium, “H” for hard, and “X” for eXtreme, respectively.

Setting up a game

To input a game from other sources, press “New Game” button, and then, input numbers as follows.

Inserting numbers

There are two ways to insert numbers on the board, determining either cell first or number first.

Assigning a cell first

Click a cell where you want to insert a number. A number panel shown below will pop up, and then, click a number.

Assigning number first

Click a number in the number pane in the “Helper” panel to set the number.

Then, click a cell you are going to assign the number.

Set a game

After you set all numbers for a game, click the “Set & Play” button in the “Game Control” panel to start the game.

Using helper functions

Helper functions

There four functions to help you solving a game.

  1. Input Check Function
  2. Single Number Function
  3. Candiates Function
  4. Marker

Input Check

The input check has two functions, duplicates check and incorrect insertion check.

“Duplicates Check” alerts when you accidentally insert a number twice in a row/column/block.

“Incorrect Insertion Check” alerts when you insert a number different from the solution, which the helper program automatically solves at the beggining (not all games are solvable by the program, though).

Singlue Number

When you want to focus a single number, you can shadow other numbers by pressing the “Single Num” button.

Initially, the number is set to one. You can change the number by pressing a number (a radio button) in the number pane.

Single Number function has four levels.

  1. Numbers other than the selected number will be shadowed.
  2. Cells inhibited by the selected number will be shadowed.
  3. Highlight selectable cells
  4. Cells inhibited by the arrangement of the selected number in other blocks will be shadowed.

Candidate function

Candidate numbers are shown in each cell.

You can delete a number that you think it is not a candidate by clicking the number.

You can also revive the number by clicking the shadowed number.

The candidate function has seven levels.

  1. Present all candidates including numbers inhibited by the numbers on the board.
  2. Present all candidates excluding numbers inhibited by the numbers on the board.
    *) In this level setting, when you assign a number to a cell, the number in the related cells will be deleted from the candidates. If you do not want the automatic deletion, set the level down to L0.
  3. Highlight selectable numbers.
  4. Numbers inhibited by the arrangement of the number in other blocks will be shadowed.
  5. Hightlighting a combination of numbers exclude other numbers (“Reserved” in the definition by Nikoli) and shadowing numbers excluded by the exclusive candidates.
  6. Highlighting a case that a number aligned in a rectangle and shadowing the number in cells inhibited by the rectangle alignment.
  7. Hightling a case that an assumption leads to no solution (a kind of proof of contradiction) and shadowing the number denied by that.

Combining the two functions

While one function (either Single Number or Candidates) is active, pressing the ohter function button will activate both functions.

Multiple Numbers

While the combined function (both Single Number or Candidates) is active, pressing the Single Number button will activate Multiple Number function, which is an experimental function.

Marker Functions

You can take note to show a number can only be in a specific row (or column) in a block with the marker function.

When the marker function with the candidate function level 1 (or above), exclusive candidates will be highlighted.

Control Buttons

You can control the state of the board with the control buttons such as stepping back or stepping forward your operations or helper functions.

Some functions may not be available depending the helper level.

  1. Rewind
  2. Backward
  3. Forward
  4. Fast Forward
  5. Select Selectables
  6. Erase Colors

Rewind

You can go back the point that you took a mistake. Otherwise, the function is the same as “Restart”.

Backward

You can go back the previous state if you want to cancel your action or the helper function.

Forward

You get another hint by clicking the “Forward” button.

Fast Forward

You get all the hints on the current helper level.

You may want to skip too easy processes (a low level manipulations such as level 2) with the function.

Select selectables

Assign numbers for selectable cells.

Erase colors

You can erase colors to highlight exclusive candidates.

Skip ineffective exclusives

Not showing candidates that meets a condition as an exclusive candidates without any effects to other numbers or other cells.

On by default (non-display). You might learn what kinds of alignment can be exclusive to others by showing all exclusive candidates.

Other Functions

New Game

To set up and play a new game, press the “New Game” button.

Restart

If you want to restart the game from the beggining, press the “Restart” button.

Change language

You can change the language from English to Japanese by pressing the “日本語” button.
